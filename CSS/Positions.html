Position in CSS

  1. Static: the original position of an element
  2. Realtive: Element is positioned Realtive to it's original/normal position. Same as static positioning, just that here we can move the element by adding offset value. Elements stays in normal document flow
  3. Absolute: Element is positioned relative to its parent. if parent is not present it will consider body tag as parent. Element is removed from the normal document flow, this means no space is created for the element. Absolute will work with parent only when parent position is relative.
  4. Fixed: Element is always positioned relative to the viewport. Element always stays in the same position even on scrrolling. Element is removed from normal document flow
  5. Sticky: Element is positioned to the normal document flow Till the element hits the top, it has normal position, after it touches the top it's position becomes fixed.

Normal Document Flow â€” Summary

Normal document flow is the natural way elements appear on a webpage:

  • Elements come one after another, top to bottom.
  • Block elements stack vertically.
  • Inline elements sit next to each other.
  • Each element takes its own space and pushes the next element down.

In short:
Normal flow = elements behave normally and follow the natural layout.
Removed from flow = element is taken out, floats freely, and does NOT take space.

Positions that stay in normal flow

  • static
  • relative
  • sticky (before sticking)

Positions removed from normal flow

  • absolute
  • fixed

One-line definition:
Normal document flow is the default arrangement of elements on the page where each element takes space and affects the others.
